In this section we will show you the **`flag`** endpoints, its parameters, combinations and specific details.\
At the moment, <a href="/" class="no-underline">Countries Sundries</a> covers a total of coats of arms of 164 countries, including different international organizations.
Since it is so similar to the `coat-of-arm` endpoint, there will be almost no change with respect to that section.

## Endpoint response and parameters

The endpoint `flag` will return the information of the countries with limit of 30 flags of countries.

## `flag/all`
The `flag/all` endpoint, if no limit is specified, will by default return 30 flags of countries from the database.

#### `id`
> _Optional Integer_\
Allows you to filter registers by id
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?id=1' -H 'x-countries_sundries_api-key: YOUR_API_KEY' -H 'x-api-version: 1' -H 'Content-Type: application/json'
```

#### `name`
> _Optional String_\
Allows you to filter registers by name
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?name=argentina' -H 'x-countries_sundries_api-key: YOUR_API_KEY' -H 'x-api-version: 1' -H 'Content-Type: application/json'
```

<GlobalQueryNote />

#### `type`
>_Optional String_\
Allows you to filter registers by type, for now only the image of type [SVG](https://developer.mozilla.org/en-US/docs/Web/SVG) is available
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?type=svg' -H 'x-countries_sundries_api-key: YOUR_API_KEY' -H 'x-api-version: 1' -H 'Content-Type: application/json'
```

## `flag/details`
This endpoint will allow you to obtain all the previous information but returning a single register, this will allow a much faster response if you need a single country and not many.\
All Query Parameters (except <a href="/docs/api-reference/global-query-parameters/" class="no-underline">Global Query Parameters</a>) are enabled for `details`.\
If no Query Parameter is specified, the first register will be returned.
