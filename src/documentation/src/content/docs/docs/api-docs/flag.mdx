In this section we will show you the **`flag`** endpoints, its parameters, combinations and specific details.\
At the moment, [Countries Sundries](http://countries-sundries-api.com/) covers a total of coats of arms of 164 countries, including different international organizations.

## Endpoint response and parameters

The endpoint `flag` will return the information of the countries with limit of 30 flags of countries.

## `flag/all`
The `flag/all` endpoint, if no limit is specified, will by default return 30 flags of countries from the database.

### Query Parameters
In the Query Parameters sections we will use the [CURL](https://curl.se/) command to show how to use parameters in the request. These query parameter tests will also be possible in [Postman](https://www.postman.com/), [Insomnia](https://insomnia.rest/) or other API testing tools, for example.

#### `id`
> _Optional Integer_\
Allows you to filter records by id
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?id=1'
```

#### `name`
> _Optional String_\
Allows you to filter records by name
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?name=argentina'
```

:::note[In case you haven't arrived yet!]
See the [Global Query Parameters](/docs/api-reference/global-query-parameters) section to discover the Query Parameters that are globally accepted for endpoints ending in `/all`!
:::

#### `type`
>_Optional String_\
Allows you to filter records by type, for now only the image of type [SVG](https://developer.mozilla.org/en-US/docs/Web/SVG) is available
```bash
curl -X GET 'http://countries-sundries-api.com/flag/all?type=svg'
```

## `flag/details`
This endpoint will allow you to obtain all the previous information but returning a single register, this will allow a much faster response if you need a single country and not many.\
All Query Parameters (Except [Global Query Parameters](/docs/api-reference/global-query-parameters)) are enabled for `details`.\
If no Query Parameter is specified, the first register will be returned.
