---
title: 'Get API-Key access'
---

This section will show you how to get your API key to access the information provided by Countries Sundries. To do this, you will first need to register with Countries Sundries.

The available registration options are:
+ Manual registration: Name, E-Mail and password
+ Auto-sign-in: Sign in with Google

## Manual registration
After registering you will be given your key to access the API information, this must be placed in the header 'x-countries_sundries_api-key', with the value of the word 'Bearer' and followed by your key.
