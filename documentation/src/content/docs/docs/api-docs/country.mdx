In this section we will show you the **`country`** endpoint, its parameters, combinations and specific details.\
At the moment, <a href="/" id="link" class="no-underline">Countries Sundries</a> covers a total of 160 countries and different international organizations.

## Endpoint response and parameters

This endpoint will return the information of the countries with limit of 30 registers.

## `country/all`
The `country/all` endpoint, if no limit is specified, will by default return 30 countries from the database.

### Example Request

<CodeExample endpoint="country/all" />

:::note[Keep in mind...]
  The fewer countries registers you bring in, the faster the server will respond.
:::

## `country/details`
This endpoint will allow you to obtain all the previous information but returning a single register, this will allow a much faster response if you need a single country and not many.\
All Query Parameters (except <a href="/docs/api-reference/global-query-parameters/" id="link" class="no-underline">Global Query Parameters</a>) are enabled for `details`.\
If no Query Parameter is specified, the first register will be returned.
